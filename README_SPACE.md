# Customer Sentiment Analysis Demo

Interactive demo for analyzing customer review sentiment using a fine-tuned DistilBERT model.

## Model

- **Model:** [IberaSoft/customer-sentiment-analyzer](https://huggingface.co/IberaSoft/customer-sentiment-analyzer)
- **Dataset:** [IberaSoft/ecommerce-reviews-sentiment](https://huggingface.co/datasets/IberaSoft/ecommerce-reviews-sentiment)
- **Accuracy:** 90.2% on test set

## Usage

Enter any customer review text and get instant sentiment analysis with confidence scores.
